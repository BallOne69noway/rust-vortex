# 🌑 RUST VORTEX DASHBOARD

**Rust Vortex** — это минималистичный и эстетичный трекер статистики для игры Rust. Проект демонстрирует интеграцию со Steam Web API и современный подход к интерфейсам.

## 🛠 Технологии (Tech Stack)
- **Framework:** React + Vite
- **Styling:** Tailwind CSS (Dark Mode, Glassmorphism)
- **API:** Steam Web API via Custom Resolution
- **Proxy:** CORS Handling for secure data fetching

## ⚡️ Функции
- [x] Автоматическое разрешение Steam Vanity URL (использование буквенных ссылок)
- [x] Синхронизация профиля в реальном времени (Ник, Аватар)
- [x] Боевая статистика: Kills, Deaths, Headshots
- [x] Трекинг игрового времени (Total Hours)
